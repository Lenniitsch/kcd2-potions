# Kingdom Come: Deliverance II – Potion Brewing Guide & Interactive Maps

> **KCD2 Alchemy Helper** – A complete potion recipe database and interactive Kuttenberg maps for Kingdom Come: Deliverance 2

Looking for an easy way to brew potions in **Kingdom Come: Deliverance 2**? This tool helps you find all KCD2 potion recipes, filter by ingredients, and navigate Kuttenberg with zoomable annotated maps.

## 🌐 Live Demo

**[duckoverflow.com/tools/kcd2/](https://duckoverflow.com/tools/kcd2/)**

## ℹ️ About

A personal hobby project built for quick recipe lookups during playthroughs. Designed as a second-screen companion — open it on your phone, tablet, or side monitor while brewing in-game.

**No claim to be complete or perfect — just good enough for my needs.**

## ✨ Features

### 🧪 KCD2 Potion Database
- **27 Potion Recipes** — Complete with effects, prices, ingredients, and step-by-step brewing instructions
- **Ingredient Filter** — Select the ingredients you have and instantly see what you can brew, with incompatible ingredients automatically greyed out
- **Search** — Find potions by name or effect description
- **8 Sort Options** — Sort by name, category, price, or ingredient count (ascending/descending)
- **Grid / List Layout** — Switch between compact grid cards or detailed list view
- **Color-Coded Steps** — Each recipe step is categorized (base liquid / ingredient / cooking / finishing) with distinct colors for quick scanning while brewing
- **Shareable URLs** — Filter state is serialized to URL params so you can bookmark or share specific searches

### 🗺️ Kuttenberg Maps
- **District Map** — Navigate Kuttenberg's neighborhoods
- **Underground Tunnel Map** — Find your way through the tunnel system
- **Zoomable & Pannable** — Powered by Zoomist with touch and mouse wheel support

### 📱 Quality of Life
- **Multi-Language** — German, English, and Italian UI + translations
- **Dark / Light Theme** — Dark-first with a toggleable light mode, persisted across sessions
- **PWA Support** — Install as a standalone app for offline use while gaming
- **Responsive Design** — Works on phone, tablet, or second monitor
- **Privacy-Friendly** — No tracking, no cookies, no analytics

## 🏗️ Tech Stack

Plain HTML, precompiled Tailwind CSS v4, and vanilla ES modules. No framework, no bundler, no npm dependencies at runtime.

## ⚠️ Notes

- All potion effects and prices are based on **max brewing skill level**
- I play in German, so the English and Italian translations might not be 100% accurate to the in-game text

If you spot any mistakes, feel free to open an issue!

## 🚀 Usage

Open `index.html` in a browser, deploy to any static host, or use the live version at **[duckoverflow.com/tools/kcd2/](https://duckoverflow.com/tools/kcd2/)**
